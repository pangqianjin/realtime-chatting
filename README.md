# realtime-chatting
基于React，antd-mobile，Node JS，socket.io和MongoDB的实时聊天应用

使用creat-react-app创建，目前优化空间比较大

# 如何启动
>yarn start //或者npm start

// 启动后端
>cd server
node app.js

# 数据库的配置
默认使用 'mongodb://localhost:27017/local'
